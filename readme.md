Nike Run Club exports often contain broken or inconsistent TCX data, which leads to incorrect distances and poor imports into platforms like Strava.

This tool fixes those issues by:
- Recalculating and correcting distance data
- Handling both GPS and non-GPS runs
- Generating clean, human-readable activity names
- Enriching metadata (pace, location, run type)
- Batch processing entire run histories

The result is a clean, consistent, and visually polished running history.
